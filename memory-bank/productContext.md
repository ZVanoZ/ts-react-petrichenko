# Product Context

- **Тип проекта**: учебный репозиторий (конспект + практические задания по курсу «Полный курс по Typescript + React»).
- **Целевая аудитория**: сам автор (и потенциально другие разработчики), проходящие курс и желающие:
  - закрепить базу по TypeScript;
  - освоить связку React + TypeScript на практических примерах;
  - иметь удобный навигационный индекс по урокам и своим решениям.

### Структура на уровне разделов

- `10-intro` — вводные материалы по курсу.
- `20-ts` — раздел **«Typescript. Базовые знания»**:
  - Содержит подразделы `10-base`, `20-necessery`, `30-generics-manipulation`, `40-classes`, `50-decorators-and-configs`.
  - В каждом подразделе — папки по урокам + `readme.md`/`index.ts` с заметками и кодом.
- `30-practice` — раздел **«React + Typescript. Большая практика»**:
  - Здесь планируется/ведётся практическая часть по React + TS.

### Как этим пользоваться

- За основу навигации брать корневой `readme.md` — он связывает номера уроков, директории и внешние ссылки.
- Для каждого нового урока:
  - создавать отдельную папку внутри соответствующего подраздела;
  - фиксировать в локальном `readme.md` только ключевые моменты и решения, не переписывая целиком теорию.
- Для помощи от AI:
  - формулировать запросы кратко и ссылаться на конкретный раздел/урок (например: «20-ts/30-generics-manipulation/050-generics-classes — перепроверить типизацию»).

### Ограничения и контекст актуальности

- Курс даёт **устойчивую базу**, но не покрывает все новейшие возможности TypeScript 5.x и React 18+ (Server Components, продвинутый роутинг и др.).
- Репозиторий целесообразно дополнять пометками, где современный подход отличается от показанного в курсе (например, новые паттерны работы с React hooks или TS utility-типами).
